#  Hotelz app 
This is a hybrid mobile application that was developed in ionic 2 for _desarrollo de aplicaciones empresariales_ course.

##  Requirements
That version or above
```bash
$ npm -v
3.10.10

3.12.0
```

##  Installation
```git
git clone https://github.com/yoinergomez/dae_hotelz_ionic.git
cd dae_hotelz_ionic
npm install
```
